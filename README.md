# F1  Project AluraDev Google
## TOP 50 Melhores projetos da Imersão !

Este é um projeto de frontend, desenvolvido como parte do curso da Alura, que exibe o grid de pilotos da temporada 2025 da Fórmula 1. Ele apresenta os pilotos em cards interativos, permitindo a busca por nome, equipe, país ou tags. O projeto conta com um design moderno e responsivo, utilizando animações suaves da biblioteca GSAP e um arquivo data.json para armazenar as informações dos pilotos.

## Objetivo do Projeto
O F1 Project AluraDev Google é uma aplicação web interativa desenvolvida com HTML, CSS e JavaScript para exibir o grid completo de pilotos da Fórmula 1 na temporada de 2025. O projeto utiliza um arquivo JSON como banco de dados local para carregar as informações dos pilotos e apresenta os dados em um formato de cards estilizados.

---

## Tecnologias Utilizadas
- HTML5: Estrutura semântica da página.
- CSS3: Estilização e layout, incluindo fontes personalizadas (Orbitron e Roboto).
- JavaScript: Lógica de busca, renderização de cards e manipulação do DOM.
- GSAP: Biblioteca de animação para efeitos visuais.
- JSON: Armazenamento dos dados dos pilotos.

## Funcionalidades
- Listagem de pilotos: Exibe todos os pilotos em cards com foto, nome, equipe e número de pódios.
- Busca dinâmica: Permite a filtragem instantânea de pilotos por nome, equipe, país ou tags.
- Estatísticas dinâmicas: Mostra o número total de pilotos, equipes e pódios combinados.
- Animações: Utiliza a biblioteca GSAP para adicionar animações de entrada elegantes aos elementos da página.
